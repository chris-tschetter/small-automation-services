# Portfolio Sample — CSV Cleanup

This is a no-spend proof asset for small fixed-scope data cleanup jobs.

Buyer-facing angle:
"Send me your messy CSV/Excel export. I will clean it, remove duplicates, normalize fields, and return a clean file plus a short report."

Files:
- `messy_contacts.csv` — intentionally messy input.
- `clean_contacts.py` — repeatable cleanup script.
- `cleaned_contacts.csv` — output file.
- `cleanup_report.md` — proof report.
